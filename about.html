TKC Programming Club

Who We Are.

The King's College Programming Club exists to provide students with a community which makes the learning and practice of programming more accessible for students. Similar to mathematics, learning how to code is best done through practice and tests. The club is made up of members who are interested and devoted to learning how to code and program, and apply those skills to applicative use such as web design.

What We Do.

To help facilitate learning code, we regularly meet to practice reading and writing code without a reference sheet. Relying on a reference sheet will hinder, if not halt, true learning of how to write code. As new members join our club, we encourage them to use resources such as Code Academy to learn the basics, then practice applying the knowledge through user-designed goals and challenges.

Why Learn Code?

Learning how to code is very beneficial for anyone's career, business, and personal satisfaction. Building your own website or blog is as satisfying as building as cooking an exquisite meal or a painting that reflects who you are and how you think. Coding unleashes the creative and productive traits inherent in everyone. It is like building a structure out of Legos, but never running out of blocks or being restricted by the size or shape of the block. Coding is limitless in what you can create or design.
